Python assignment demonstrating JSON creation, parsing API responses, and handling nested JSON data.

Task 1 — Build a JSON Structure:
- Created a Python dictionary representing a user profile.
- Fields included: name, age, email, is_active, and skills.
- Converted the dictionary into a JSON string using `json.dumps()`.
- Printed the JSON with proper indentation.

Task 2 — Parse an API Response:
- Parsed a mock API response provided as a JSON string.
- Extracted and printed:
  - Username
  - Score
- Displayed the message:
         - User alex99 scored 87.5 points

Task 3 — Handle Nested JSON:
- Worked with nested JSON data containing user address details.
- Extracted and printed:
- City
- Zip code
- Added a new key `"country": "India"` inside the address object.
- Printed the updated JSON structure.
